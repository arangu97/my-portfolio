# Iñigo Aranguren · Web Personal

Este repositorio contiene mi página web personal y portfolio profesional.

La web está orientada a mostrar, de forma simple y directa:

- Mi perfil profesional como Tech Lead de Producto.
- Mi experiencia laboral y académica.
- Un portfolio general con casos/proyectos destacados.
- Un blog con publicaciones cortas sobre tecnología, producto y liderazgo.

## Estructura

- `index.html`: estructura principal de la web.
- `styles.css`: estilos minimalistas.
- `script.js`: contenido dinámico básico (tarjetas de portfolio y blog).

## Publicación

Este proyecto está pensado para publicarse en GitHub Pages.

## Idioma

El contenido está actualmente en español.
